    - #什么是丁达尔现象#？19世纪，英国物理学家约翰·丁达尔发现，当光线穿过某种胶体时，由于微粒的散射作用，可以看到胶体里出现一条条光亮“通路”，这种光学现象就是丁达尔现象。
